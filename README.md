## Project Description

This project implements a Smart Device Management System using Object-Oriented Programming in Python.

The system demonstrates:

- Classes and Objects
- Inheritance
- Encapsulation
- Constructors
- Getters and Setters
- Menu-driven Programming
- Functions
- Conditional Statements
- Loops

## Devices

- Temperature Sensor
- Smart Light
- Security Camera

## How to Run

1. Open the project in PyCharm.
2. Run main.py.
3. Follow the menu displayed in the terminal.
